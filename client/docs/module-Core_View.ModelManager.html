Class: ModelManager

Core/View.ModelManager

Container for all the models which affect how the final rendering outcome looks. Cameras, lights, spheres and slicers.

Constructor

new ModelManager()

Constructs a new model manager, will initiate models for 'GLOBAL' and cell 0 by default.
