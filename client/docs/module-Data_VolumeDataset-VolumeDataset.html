VolumeDataset

Data/VolumeDataset~ VolumeDataset

Represents one volume dataset, holds the isovalues, gradient, curvature etc. Most of the heavy CPU calculations should be done @ server side though it is possible to do most of it in the browser.

Constructor

new VolumeDataset(isovalues)

Constructs a new volume dataset
Parameters:

isovalues Int16Array the isovalues, scaled from the range [0, 4095] to [0, 2^15]
