Class: LinkGrouper

View~LinkGrouper

Groups properties together. This is used for linking properties such as lighting, camera etc between views.

Constructor

new LinkGrouper(maxGroups)

Creates a new link group. The "master cell" of each group is the cell ID at index 0 in the given group.
Parameters:
Name Type Description
maxGroups number the maximum amount of link groups allowed

Methods

getMasterCellID(cellID) → {number}

Gets the master cell for this cellIDs group. If this cell is ungrouped, it will simply return the same cellID as the input
Parameters:
Name Type Description
cellID number the cell ID
Returns:
the master cell ID of this cell IDs group (if any)

number
