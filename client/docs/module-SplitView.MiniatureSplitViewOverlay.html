Class: MiniatureSplitViewOverlay

SplitView.MiniatureSplitViewOverlay

Makes an overlay linked to a split view.

Constructor

new MiniatureSplitViewOverlay(config)

Constructs a new miniature split view overlay. This class will use the layout of a module:SplitView.MiniatureSplitView object and be linked to it in some ways. The purpose of this overlay is to record events for each subview and pass them on to the according event handler, and to render the layout in as simple as possible manner.
Parameters:
Name Type Description
config module:SplitView.MiniatureSplitViewOverlayConfiguration Configuration of the overlay
