MiniatureSplitViewConfiguration

Configuration options for a miniature split view
Type:
  • Object
Properties:
Name Type Description
divID string the html ID of the div to insert the split view into
aspectRatio number the width/height ratio of this split view
state string the initial state, one of the following: 'ADD', 'REMOVE', 'LINK-ADD', 'LINK-IN-PROGRESS'.
canAddRemove bool Whether or not the view can add or remove cells. (Must correspond with the state parameter)
showIDs bool Whether or not to display cell IDs
bottomTopThresholdPercentage number range [0, 1], The percentage threshold (of the row height) at which the view cursor position snaps to being seen as IN BETWEEN two rows instead of on one.
colors Array.<string> An array of colors to use for linking - to distinguish between multiple link groups. Only needed for linking. The length of this array will be the max amount of link groups.

options Object customizable options (TODO fill in options)
Properties
Name Type Description
showIDs bool true to show the cell ID in the upper left corner
bottomTopThresholdPercentage number [0,1] Specifies the percentage at which the selection should snap to be seen as "between" two rows. Useful for resizing views etc. NOTE: The row spacing will be the same as the column spacing when it comes to resizing.
subcellLayout module:Widgets/View.SubcellLayout the layouting of the subcells
listener function The event listener to delegate mouse events etc to. Arguments: (cellID, subcellName, event)
