RenderingStep

Configurations for one rendering step.
Type:
  • Object
Properties:
Name Type Description
programInfo twgl.ProgramInfo Holds the shader program (see twgl library docs)
frameBufferInfo twgl.FramebufferInfo Holds the frame buffer, pass in null to set target to screen. (see twgl library docs)
bufferInfo twgl.BufferInfo Holds the shape to render, usually the bounding box (see twgl library docs)
uniforms Object JSON object corresponding to the uniforms in the shader.
glSettings module:Core/Renderer.GLConfiguration specify a set of openGL settings as key-value pairs in a JSON. These settings will be applied before the rendering is done.
