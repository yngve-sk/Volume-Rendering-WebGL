RailInfo

Represents a rails underlying structure, only used in SlicerModel. The rail is a line from p0 to p1
Type:
  • object
Properties:
Name Type Description
p0 twgl.primitives.v3
p1 twgl.primitives.v3
direction string X, Y or Z, axis the rail is aligned with
translate Array.number size is always 2. Direction the rail is to be translate in normal to its direction axis, ex if direction is X then it is to be translated along the Y by offset translate[0] and the Z axis by offset translate[1].
