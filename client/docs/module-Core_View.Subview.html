Class: Subview

Core/View.Subview

Represents a subview, will manage the renderers for each subview, and also hold pointers to the models which the renderers will be drawing. The reason for holding pointers is it makes linking different proprties together easier.

Constructor

new Subview(webGLContext, models)

Initializes a new subview
Parameters:
Name Type Description
webGLContext WebGL2Context The webGL context to render to
models Object JSON with the initial models
