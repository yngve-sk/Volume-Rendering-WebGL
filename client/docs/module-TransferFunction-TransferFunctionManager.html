Gets the cell ID referenced by the TF editor
Parameters:
Name Type Description
tfEditorKey string 'GLOBAL' or 'LOCAL'
Returns:
- if 'GLOBAL', will simply return 'GLOBAL' since the global TF editor is always editing its own TF, otherwise it'll return an actual cell

string | number

notifyDiscreteTFDidChange(key, textureBounds)

Notify the TF manager that the transfer function did change. Will be called from the TF editor any time the texture does change
Parameters:
Name Type Description
key string the transfer function key, 'LOCAL' or 'GLOBAL'
textureBounds module:TransferFunction.TextureBounds The texture bounds, passed in as a convenience. Will be returned when views get the texture bound via getCanvasForTFKey

notifyTFPointsToCanvasWithID(key, ngid)

Called each time a TF editor is created. Called after the TF editor is done compiling, will link the canvas to the TF ID.

Name Type Description
key string the TF key, 'LOCAL' or 'GLOBAL'
ngid number the angular $scope$id of the view, used to identify it
