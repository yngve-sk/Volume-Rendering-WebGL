LinkerAndSplitterViewSettings

Settings for the linker / splitter views.
Type:
  • Object
Properties:
Name Type Description
maxRows number Maximum amount of rows allowed (Note: Will decide amount of subviews available)
maxColumns number Maximum amount of columns allowed (Note: Will decide amount of subviews available)
showIDs bool if true, the cell IDs will show
bottomTopThresholdPercentage number Percentage at which the mouse snaps to being seen as being closest to the bottom/top of the row as opposed to closest to one of the sides of the current cell. Only used when adding cells.
colors Object Colors of the view
Properties
Name Type Description
LINKS Array.<string> Array of valid CSS colors (hex, rgba, string). The length of this array will be the maximum amount of groups allowed, and will also decide the coloring of the link groups in each link view.
REMOVE string Color of a cell when mouse is over it and the mode set to REMOVE (remove cells).
ADD string Color of a cell when mouse is over it and the mode set to ADD (add cells).
