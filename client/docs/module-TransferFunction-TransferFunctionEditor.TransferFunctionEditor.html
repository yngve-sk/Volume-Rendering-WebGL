Parameters:
Name Type Description
displayOptions module:TransferFunction.DisplayOptions
getInteractionMode function Function to fetch the current interaction mode as set in the GUI. Will return 'Select' or 'TF'.
$scope$id number Angularjs ID of the view this editor will be embedded into, this ID is used as a key to communicate with the environment
