IsovalueHistogram

Represents a histogram of isovalue counts. The length of this array will always be 4096. The values will be in the range [0, 2^15]. (Would need to use Int32Array is value-counts exceed 2^15)
Type:
  • Int16Array

SelectionInfo

Information about a selection, contains an index buffer + all available selections.
Type:
  • Object
Properties:
Name Type Description
indices Array.<uint> Indices of the selections for this cell ID.
selections Array.<Object> All selections made by all views (could possibly be useful for more advanced features).
