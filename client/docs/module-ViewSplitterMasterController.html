Module: ViewSplitterMasterController

Bundles together all the module:Widgets/View.MiniatureSplitView widgets, is responsible for the communication between the miniature split view widgets and the environment. Provides an easy way for the environment to fetch the information it needs from these widgets, such as link groupings and information about the current layout. Will also notify listeners of certain events, such as when the layout changes.
