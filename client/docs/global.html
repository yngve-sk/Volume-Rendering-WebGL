BUILTIN_PROGRAMS

--- --- HOW TO ADD A SHADER PROGRAM: --- 1. Add the folder to the BUILTIN PROGRAMS as Foldername: 'Foldername' --- 2. Register the vertex and fragment shader with glslify + file path --- Shader program should now be available by calling getProgramInfo(name) --- --- FOLDER STRUCTURE: Foldername/shader._.glsl , _ = v or f, or whatever other type

translateCylinderVerticesToEdge(alongAxis, direction, offset) → {twgl.m4}

Parameters:
Name Type Description
alongAxis Object axis the cylinder is centered and aligned along
direction Object direction to translate it 2D-wise. ex if aligned along axis X, this will represent how much to translate it by offset along the Y (index 0) and Z (index 1) axis
offset Object how far to translate the cylinder vertices. If creating a bounding box translate by size/2
Returns:
translation the translation matrix
